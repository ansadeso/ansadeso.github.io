Задача 1.8

Від бакена, який знаходиться по середині широкої річки, відпливло два човни, A і B. Обидва човна почали рухатися по взаємно перпендикулярним прямим: човен A - вздовж річки, а човен B - поперек. Відпливши на однакову відстань від бакена, човни повернулися назад. Знайти відношення часу, руху човнів `\tau_A/\tau_B`, якщо швидкість кожного човна відносно води в `\eta = 1,2` разів більше швидкості течії.

Дано:
`S_1 = S_2 = S`
`\eta = V^'/V_0 = 1,2`
`\tau_A/\tau_B` - ?
Розв`язання

`AB = S_2`

`AC = S_1`

`t_B = (2S_2)/V_B = (2S_2)/(sqrt(V^('2)-V_0^2))`

`t_B = (2S)/(sqrt(V^('2)-V_0^2))`

`t_A = S_1/(V^' + V_0) + S_1/(V^'- V_0)`


`t_A = S/(V^' + V_0) + S/(V^'- V_0)`

`t_A = (S(V^'- V_0))/((V^' + V_0)(V^'- V_0)) + (S(V^' + V_0))/((V^'- V_0)(V^' + V_0))`

`t_A = (S(V^'- V_0 + V^' + V_0))/(V^('2)- V_0^2) `

`t_A = (2SV^')/(V^('2)- V_0^2) `

`\tau_A/\tau_B = t_A/t_B = (2SV^')/(V^('2)- V_0^2) * (sqrt(V^('2)-V_0^2))/(2S)`

`\tau_A/\tau_B = (V^')/(sqrt(V^('2)-V_0^2))`

`\tau_A/\tau_B = (V^')/(V^'sqrt(1-V_0^2/V^('2)))`

`\tau_A/\tau_B = (1)/(sqrt(1-V_0^2/V^('2)))`

`\tau_A/\tau_B = (1)/(sqrt(1-(V_0/V^')^2))`

`1/\eta = V_0/V^'`

`\tau_A/\tau_B = (1)/(sqrt(1-(1/\eta)^2))`

`\tau_A/\tau_B = (1)/(sqrt(1-1/\eta^2))`

`\tau_A/\tau_B = (\eta)/(sqrt(\eta^2-1))`

`\tau_A/\tau_B = (1,2)/(sqrt(\1,44-1))` ≈ ` 1,8`

Відповідь: `\tau_A/\tau_B = (\eta)/(sqrt(\eta^2-1))` ≈ ` 1,8`.
