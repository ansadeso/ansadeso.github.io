Точка пройшла половину шляху зі швидкістю `V_0`. На решті шляху вона половину часу рухалася зі швидкістю `V_1`, а останю частину шляху пройшла зі швидкістю `V_2`. Найти середню, за весь час руху, швидкість точки.

Дано:
`V_0`
`V_1`
`V_2`
`S_0 = S_1 + S_2`
`S/2 = S_1 + S_2`
`t_1 = t_2`
`\ltV\gt` - ?
Розв`язання

`S=\intVdt`

`S=Vt`

`S=S_0+S_1+S_2`

`S=2S_1 + 2S_2`

`2S_1 + 2S_2 = S_0 + S_1 + S_2`

`2V_1t_1 + 2V_2t_2 = V_0t_0 + V_1t_1 + V_2t_2`

`V_1t_1 + V_2t_2 = V_0t_0`


`t_0 = (V_1t_1 + V_2t_2)/V_0`

`t_0 = (t_1(V_1 + V_2))/V_0`

`S_1 + S_2 = S/2`

`V_1t_1 + V_2t_2 = (\ltV\gt(t_0 + t_1 + t_2))/2`

`t_1(V_1 + V_2) = (\ltV\gt(t_0 + 2t_1))/2`

`2t_1(V_1 + V_2) = \ltV\gt(t_0 + 2t_1)`

`\ltV\gt = (2t_1(V_1 + V_2))/(t_0 + 2t_1)`

В друге рівняння підставляємо перше.

`\ltV\gt = (2t_1(V_1 + V_2))/((t_1(V_1 + V_2))/V_0 + (2t_1V_0)/V_0)`

`\ltV\gt = (2t_1(V_1 + V_2))/((t_1(V_1 + V_2)+2t_1V_0)/V_0)`

`\ltV\gt = (2t_1V_0(V_1 + V_2))/(t_1(V_1 + V_2)+2t_1V_0)`

`\ltV\gt = (2t_1V_0(V_1 + V_2))/(t_1(V_1 + V_2 + 2V_0))`

`\ltV\gt = (2V_0(V_1 + V_2))/(2V_0 + V_1 + V_2)`

Відповідь: `\ltV\gt = (2V_0(V_1 + V_2))/(2V_0 + V_1 + V_2)`
