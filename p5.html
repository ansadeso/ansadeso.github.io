Компілювання програми

У Visual Studio Code пишемо наступну програму з 'Hello World'. Потрібно створити файл з розширенням ASM. Назва має бути короткою. На фото файл HELLO1.ASM. ASM розширеня асемблера.
Далі необхідно зберегти ти скомпілювати нашу програму. Компілюваня це перетвореня програми в машиний код. Для компілюваня будемо використовувати nasm з командного рядка. В Windows у пошуку введемо "cmd" без лапок. Відкриваємо консоль. Вона матиме приблизно наступний вигляд.
Наступним кроком буде перейти, в консолі, до каталогу де знаходиться наш файл HELLO1.ASM. Для зміни каталогу, в консолі, використовуємо команду cd.
Натискаємо Enter та опиняємося в потрібному каталозі.
В даному каталозі повинен знаходитися nasm компілятор. Його вигляд приблизно такий.
Для компілюваня вводимо наступну команду "nasm hello1.asm" без лапок.
Натискаємо Enter. Після чого отримуємо двійковий файл в машиному коді.
Щоб відкрити цей файл в Visual Studio Code необхідно встановити розширеня HexEditor.
Далі права кнопка миші. Open with Code.
Попередженя.
